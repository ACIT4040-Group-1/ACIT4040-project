# Setup on the eX3 cluster:
## First time set-up:
```
sbatch setup_stylegan3.sh
```

# How to use:
- Modify the n_images variable in generate_images.sh to your desired number of images.
- Run the script: 
```
sbatch generate_images.sh
```
